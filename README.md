# WarehouseDB

WarehouseDB is a warehouse management system that combines a PostgreSQL-based backend with a C# Windows Forms application for storing, querying, and managing operational data. The project models warehouses, goods, customers, factories, and transactions, and uses techniques such as SQL queries, stored procedures/functions, table inheritance, and Npgsql-based integration to demonstrate practical database design and desktop application connectivity in a business-oriented setting.
